Use the sidebar or the cards below to navigate to any module. Start with Well Mapping to set up an experiment, then use the Well Explorer to review images, run predictions, and analyse results. New here? See the documentation for an overview, retraining instructions, and the project roadmap.

Well Mapping
⚙

Mapping Dashboard

Interactive Bokeh dashboard to define and edit the well plate layout for each experiment: assign drugs to source wells and map source wells to destination wells.

Open dashboard
Well Explorer
🔍

Explorer Dashboard

Browse VAST & Leica images well by well. Manually annotate fish validity, somite counts, and orientation. Run deep-learning predictions (quality, somite counting, orientation) directly from the interface.

Open dashboard
📊

Experiment Listing

Summary table of all experiments: total destination wells, mapped wells, imaged wells, VAST capture efficiency, annotated fish counts and percentage of valid fish.

View table
💊

Drug Listing

Browse all drugs by source well. Expand any row to see thumbnail images of valid and invalid fish, with somite statistics per drug condition.

View drugs
📈

Statistics

Detailed per-experiment metrics: plate fill efficiency, VAST efficiency, fish quality breakdown (all / training set / validation set), somite count statistics, and comparison of DL predictions against manual annotations (agreement rate, MAE on total & defective somites).

View statistics
▶

Drug Somite Plot

Interactive histogram explorer: select a drug derivation and one or more experiments, then plot the distribution of total and defective somite counts from DL predictions and/or manual annotations, with overlay and validity filters.

Open plot
