Pick an experiment to compare the model's predictions against manual annotations. By default the page restricts itself to the honest subset — wells flagged use_for_test=True or with no flag set at all (typical for annotations added after the model was trained). Switch Subset if you want to see how it performs on training/validation data too (those numbers are biased low — the model has seen them).

The page shows three views of the same set of wells, all linked:

  • Aggregate metrics — MAE / RMSE / bias / R² for total and defective somite counts; confusion matrix and accuracy / precision / recall / F1 for the valid flag; agreement % for orientation.
  • Plate heatmap — each well coloured by |predicted total − annotated total|. Green = good agreement, red = bad. Clusters in one corner usually mean a microscope issue rather than a model issue. Tap a cell to preview that well's image.
  • Per-well disagreement table — sortable, default order is largest |Δ total| first so the worst cases are on top. Click a row to preview the image in the right pane.
